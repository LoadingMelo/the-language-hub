## People

Like in any language, Swahili has words for people you interact with

> Fun Fact: **Rafiki** is the Swahili word for friend.

### Family

| Swahili  | Translation  |
| -------- | ------------ |
| baba     | dad/father   |
| mama     | mom/mother   |
| kaka     | brother      |
| dada     | sister       |
| babu     | grandad      |
| bibi     | grandma      |
| mjomba   | uncle        |
| shangazi | aunt         |
| mpwa     | nephew/niece |
